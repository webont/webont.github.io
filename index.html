Web Ontologies

In recent years, especially as part of the Semantic Web initiative, there has been a number of proposals for Web-oriented Ontology languages. The most prominent of these proposals is the W3C's Web Ontology Language, OWL.

Projects Hosted Here

  • The OWLED workshop series
    • There are a number of task forces
  • The Wiki of and for the OWL community
  • The OWL 1.1 specifications, submitted to the W3C by the OWLED community, now superseded by the OWL 2 recommendation.
